Estrutura Básica

O HTML se compõe de três blocos básicos. Conforme descritos abaixo:

  • HTML

    Tag que abre o bloco principal

  • HEAD

    Tag que abriga o cabeçalho, nele contém o título e os metadados da página

  • BODY

    Tag onde fica o corpo da página, tudo o que o usuário consegue ver

Tags

As tags são usadas para informar ao navegador a estrutura do site.<
A principal característica das tags é estarem sempre dentro dos sinais de “maior que” e “menor que” (<>).

Atributos

Os atributos são usados para personalizar as tags, modificando sua estrutura ou funcionalidade.

Texto

As tags de texto definem textos, estilos de fonte, parágrafos, spans, quebras de linhas, etc.
Tais como: p / span / pre / b / i / br / hr

Listas Ordenadas e Não Ordenadas

Existe duas formas de criar listas, podemos utilizar uma lista ordenada, a partir da tag "ol", ou uma lista não ordenada, a partir da tag "ul".
Posteriormente, incluímos dentro da lista os elementos da mesma, dentro da tag "li".

Links

A tag de link HTML é responsável por fazer a ligação entre um documento e outro, sendo ele da mesma página ou de uma página de outro domínio.
